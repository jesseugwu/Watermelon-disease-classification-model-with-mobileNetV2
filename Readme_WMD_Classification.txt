The codes are python files used for the development of model for watermelon disease classification. The classification is among 4 classes. Anthracnose, healthy, Downy Mildew and Healthy.


To run the code, you need jupterNote book or Google Colab,
Install Python libraries such as matplotlib, tensorflow, numpy, seaborn, sklearn.

System requirement: Minimum of 8GB RAM, 252GB Local disk, with GPU architecture of RX 5600M.

The Mobile Device Application
The Android Application is a minimum viable project (Demo) for the on-mobile watermelon disease classification.
Use the gallery or camera option to capture image of the disease. It will display the classification result, the date and time of the classification as well as the recommendation.
